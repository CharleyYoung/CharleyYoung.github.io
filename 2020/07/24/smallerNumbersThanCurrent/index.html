有多少小于当前数字的数字

发表于 2020-07-24 | 更新于 2020-07-27

记录做leetcode上1365.有多少小于当前数字的数字的心得

问题描述

给你一个数组$nums$，对于其中每个元素$nums[i]$，请你统计数组中比它小的所有数字的数目。
换而言之，对于每个$nums[i]$你必须计算出有效的$j$的数量，其中$j$满足$j != i$且$nums[j] < nums[i]$
以数组形式返回答案。
$2 <= nums.length <= 500$
$0 <= nums[i] <= 100$
