leetcode题目“报数”

发表于 2019-04-15

对leetcode题目$38.报数$问题实现的一些思路记录

问题描述

报数序列是一个整数序列，按照其中的整数的顺序进行报数，得到下一个数。其前五项如下:

1.$1$被读作$”one\ one”$,$“一个一”$，即$11$
2.$11$被读作$”two\ one”$,$“两个一”$，即$21$
3.其余项依次类推
给定一个正整数 n（1 ≤ n ≤ 30），输出报数序列的第 n 项。
注意：整数顺序将表示为一个字符串。

思路

1.仔细观察可知，整数序列的第n项是由其n-1项“翻译”得到的，这样的结构非常适合使用递归来解决问题
2.根据给出的5项可以发现一个规律，“翻译”的规则实际上就是统计连续的相同整数的个数
